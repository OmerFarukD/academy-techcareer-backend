
Proje için veri tabanında çalıştırılacak Docker komutu:

cmd Komutu:
docker volume create ecommerce_pgdata && docker run -d --name ecommerce_postgres --restart unless-stopped -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=123456 -e POSTGRES_DB=ecommerce_db -p 5433:5432 -v ecommerce_pgdata:/var/lib/postgresql/data postgres:16-alpine

PowerShell Komutu:
